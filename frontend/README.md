# SachAI - Multi-Modal AI Detection Platform

## Overview

SachAI is a full-stack AI-powered platform designed to combat digital misinformation through deepfake detection, fake news verification, and explainable AI analysis.

The platform combines computer vision, machine learning, and fact-checking services to help users assess the authenticity of digital content in real time.

---

## Features

### Deepfake Image Detection

* Custom CNN-based deepfake detection model
* Trained on 200,000+ real and fake images
* Approximately 89% detection accuracy
* Real-time image analysis

### Fake News Verification

* Claim verification using external fact-checking services
* Context-aware news analysis
* Source credibility assessment

### Explainable AI

* Grad-CAM visualizations
* Model decision transparency
* Visual explanation of detection results

### Multi-Modal Analysis

* Image authenticity verification
* News claim verification
* Unified analysis dashboard
